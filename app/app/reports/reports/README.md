# Reports Folder

This folder stores evaluation outputs generated by `project_001/main.py`.

## Structure

- `latest_run.txt`: The most recent run id.
- `runs/`: Contains one subfolder per execution.

Each run folder follows this naming pattern:

- `run_YYYYMMDD_HHMMSS/`

Each run folder contains:

- `metrics_summary.csv`: Tabular metrics for all models.
- `metrics_summary.json`: JSON version of the metrics table.
- `summary.md`: Human-readable markdown summary.
- `metadata.json`: Run metadata (dataset, timestamp, pair count).
- `roc_curve.png`: ROC comparison chart.
